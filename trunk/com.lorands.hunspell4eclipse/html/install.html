Install

Installation is trivial:

  1. Just install the plug-in as usually. (Probably already done that.)
  2. Fetch some hunspell (or myspell) dictionaries.
  3. Navigate to: Windows - Preferences - General - TextEditors - Spelling
  4. Select Hunspell4Eclipse as the spelling engine.
  5. Hit browse, select the dictionary you want to use.
  6. Done.
