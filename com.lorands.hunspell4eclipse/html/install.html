Installation and settings

Install

  1. Just install the plug-in as usually. (Probably already done that.)
  2. Fetch some hunspell (or myspell) dictionaries.
    Some URLs:
    • http://wiki.services.openoffice.org/wiki/Dictionaries
    • https://wiki.mozilla.org/L10n:Dictionaries
  3. Put the dictionaries in a specialized directory (eg. in "/Users/xxx/Library/Spelling" on a Mac).

Preferences

  1. Navigate to: Windows - Preferences - General - TextEditors - Spelling
  2. Select Hunspell4Eclipse as the spelling engine.
  3. Hit browse, select the dictionary you want to use.
  4. If the directory containing the selected ".dic" file contains also an english one, you can select the check-box allowing the english words acceptance..
  5. Set the thresolds
  6. Set the ignoring options
  7. Done.
