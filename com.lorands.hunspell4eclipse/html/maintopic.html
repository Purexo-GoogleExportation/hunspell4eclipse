About

Hunspell4Eclipse is an integration of the Hunspell spelling tool as an Eclipse spell checking engine.

Integration is done using JNA Java bindings for Hunspell developed by Flamming Frandsen.

This plug-in is capable to spell checking with any Hunspell dictionary (see a collection of links to many Hunspell dictionaries here).
